◆ get_content()

core_courseformat\local\overview\overviewitem::get_content ( )

Gets the content for this item.

Items can utilize either a renderable object or a pre-rendered string as their content.

  • For simple items, a plain string is sufficient and can be used in any context.
  • For more complex items, a renderable object is preferable. This allows the item to be rendered differently depending on the context, providing greater flexibility.
Return values
string|core::output::renderable|null
