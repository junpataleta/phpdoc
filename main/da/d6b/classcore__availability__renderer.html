core_availability_renderer::render_core_availability_multiple_messages ( core_availability_multiple_messages $renderable)

Renders HTML for the result of two or more availability restriction messages being combined in a list.

The supplied messages should already take account of the 'not' option, e.g. an example message could be 'User profile field Department must not be set to Maths'.

This function will not be called unless there are at least two messages.

Deprecated
since Moodle 4.0 MDL-716916 - please do not use this function any more.
