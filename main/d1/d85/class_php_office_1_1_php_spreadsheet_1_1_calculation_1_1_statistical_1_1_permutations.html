PERMUTATIONA.

Returns the number of permutations for a given number of objects (with repetitions) that can be selected from the total objects.

Parameters
mixed$numObjsInteger number of different objects Or can be an array of values
mixed$numInSetInteger number of objects in each permutation Or can be an array of values
Return values
array|float|int|stringNumber of permutations, or a string containing an error If an array of numbers is passed as an argument, then the returned result will also be an array with the same dimensions
