Parameters
$allowreservedDetermines whether the parameter value SHOULD allow reserved characters.

As defined by [RFC3986], these characters are :/?#[]@!$&'()*+,;= to be included without percent-encoding. This property only applies to parameters with an in value of query. The default value is false.
