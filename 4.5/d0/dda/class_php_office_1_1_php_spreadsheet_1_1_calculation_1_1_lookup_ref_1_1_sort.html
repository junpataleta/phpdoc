SORTBY The SORTBY function sorts the contents of a range or array based on the values in a corresponding range or array.

The returned array is the same shape as the provided array argument. Both $sortIndex and $sortOrder can be arrays, to provide multi-level sorting.

Parameters
mixed$sortArrayThe range of cells being sorted
mixed$argsAt least one additional argument must be provided, The vector or range to sort on After that, arguments are passed as pairs: sort order: ascending or descending Ascending = 1 (self::ORDER_ASCENDING) Descending = -1 (self::ORDER_DESCENDING) additional arrays or ranges for multi-level sorting
Return values
mixedThe sorted values from the sort range
