◆ $accuracyRadius

int null readonly int GeoIp2\Record\Location::$accuracyRadius

The approximate accuracy radius in kilometers around the latitude and longitude for the IP address.

This is the radius where we have a 67% confidence that the device using the IP address resides within the circle centered at the latitude and longitude with the provided radius.
